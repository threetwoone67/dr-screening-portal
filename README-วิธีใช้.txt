ชุดแก้ภาษา/Sidebar/Report สำหรับใช้งานจริง

ไฟล์ที่แก้:
- language.js: ระบบภาษาเดียวทั้งเว็บ Thai/English เท่านั้น
- dashboard.html: sidebar เสถียรและมีประวัติการตรวจ
- knowledge-dr.html: แปลเนื้อหาเกร็ดความรู้ได้
- retina-report-new.html: report ใช้ภาษาและ DOC ดึงรูปจาก results/image_url
- appointment-followup.html, patient-history.html, patient-transfer.html: sidebar + ภาษา sync

วิธีใช้:
1) Backup ไฟล์เดิมก่อน
2) แตก ZIP แล้ว copy ไฟล์ทั้งหมดไปทับในโฟลเดอร์ VS Code
3) ปิด Live Server แล้วเปิดใหม่
4) กด Cmd+Shift+R หรือ Ctrl+Shift+R

หมายเหตุ: ไม่ได้แก้ retina-analysis.html ตามที่แจ้งว่าใช้ได้แล้ว
